# De Slimste Mens Timer App — Project Overview

## Description

An Android app built with Delphi / FireMonkey that replaces the physical timer devices from the "De Slimste Mens" board game. The app is a countdown timer with scoring functionality.

## Functional Requirements

- Countdown timer starting at 60 seconds
- Score counts down 1 point per second while running
- Start/Stop button to toggle the timer
- +20 button: adds 20 seconds/points to the current score
- -20 button: subtracts 20 seconds/points from the current score
- Portrait-only layout (no landscape support needed)

## Visual Design

- Background: slightly darker red
- Score: displayed prominently at the top, centered
- Three buttons below the score, left to right: [+20] [Start/Stop] [-20]
- Button shape: round
- Button color: slightly orange/yellow
- Text color: white (score and buttons)

## Tech Stack

- Language: Delphi (Object Pascal)
- Framework: FireMonkey (FMX)
- Target platform: Android
